* 画面上での操作

  投稿の方法は通常の操作と異なり，投稿用の画面はありません
  画面最下部にあるテキストボックスに入力し，更新ボタンを押下することで投稿になります

  ※ テキストボックスが空の状態で更新ボタンを押下するとタブのダブルクリックと同様の処理となり，
     上部ペインで選択されている項目を取得します

  ●タイムライン
    自分と follow している人の発言を取得します

  ●返信一覧
    自分宛にされた発言 ( @ユーザ名 ) を取得します

  ●お気に入り
    Favarite 登録した ( ふぁぼった ) 発言の一覧を取得します

  ●受信メッセージ
    受信済みダイレクトメッセージを表示します

  ●送信メッセージ
    送信済みダイレクトメッセージを表示します


  下部ペインの各発言にフォーカスを当てた状態でコンテキストメニューを表示させることができます

  【全文を読む】
   新たな画面を開き，
    - 発言の全文
    - 発言者名
    - 発言時刻
    - 自己紹介
    - コメント ID
    - ユーザ ID
    - クライアント名
   を表示します

  【つぶやく】
   フォーカスを下部ペインのテキストボックスに移します

  【言い返す】
   返信用のマーク ( @ + ユーザ名 ) を付加した上でフォーカスをテキストボックスに移します

  【メッセージ送信】
   選択したユーザに向けてダイレクトメッセージを送信します


  【お気に入り登録（ふぁぼる）】
   選択した発言を Favarite 登録します

  【お気に入り削除】
   選択した発言をお気に入り一覧から削除します

  【フォローする】
   選択したユーザを follow します

  【フォローやめる】
   選択したユーザを follow から外します


  【友達のタイムライン】
   選択したユーザの発言履歴を表示します

  【友達のタイムライン(with others)】
   選択したユーザ，及びその Follow 対象のユーザの発言履歴を表示します


  【友達のホーム】
   選択したユーザのホームを開きます

  【友達のFavorites】
   選択したユーザの Favorites を表示します

  【友達のサイト】
    選択したユーザが Twitter で登録しているサイトを表示します
