* エンターキーについて

　エンターキーは項目決定に用います

　●トップ画面
　　シングルクリックで項目選択, ダブルクリックで実行になります
　　※画面によって上記と異なる動きをする場合があります
　　　以下, 特に記載がない場合はシングルクリック時の動きになります

　　【コミュニティ】タブ
　　　・コミュニティ一覧　下部ペイン
　　　　1.トピック一覧
　　　　　選択されたコミュニティのトピックを一覧表示します
　　　　2.トピック一覧 (ログ)
　　　　　上記内容のログを表示します
　　　　3.ブラウザで開く
　　　　　コミュニティトップをブラウザで表示します

　　【その他】タブ
　　　・マイミク一覧
　　　　1.「ブラウザで開く(ユーザーのページ)」
　　　　2. 「メッセージを送る」
　　　・紹介文
　　　　1.「ブラウザで開く(ユーザーのページ)」
　　　　2.紹介文を簡易表示
　　　・足あと
　　　　1.「ブラウザで開く(ユーザーのページ)」
　　　・ブックマーク
　　　　1.ブックマークしたトピックへ遷移します
　　　・お気に入りユーザー
　　　　1.「ブラウザで開く(ユーザーのページ)」
　　　　2.自己紹介を簡易表示
　　　・お気に入りコミュ
　　　　1.トピック一覧
　　　　2.トピック一覧(ログ)
　　　　3.ブラウザで開く

　　【コンテキストメニュー】
　　　シングルクリックで決定になります

　●ユーザ選択中
　　選択中ユーザのプロフィールを表示します
