　●らんらんビュー上での操作

　　ver.0.9.0.0 Beta12 よりらんらんビュー上でのグラブ＆スクロールができます
　　らんらんビュー上にマウスカーソルを合わせると，カーソルが手のひらになります
　　( 下部ペインに記事が入りきらない場合のみ )
　　このときに左クリックをするとカーソルが物を掴んだようなカーソルに変化します
　　この状態でそのまま上下に動かすことで画面のスクロールが行えます

　　また，らんらんビュー上でダブルクリックされた場合，その位置によってコメントの上下移動をします
　　表示領域を 100 として，上下 40% の領域で反応します
　　( つまり，中央の 20% 分の領域では反応しません )

　　※ Ver0.9.0.0 Beta15 時点でらんらんビュー上で Ctrl+C などのショートカットは使用できません
