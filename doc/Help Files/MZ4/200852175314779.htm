* MZ4 を使うには

  MZ4 は mixi，Twitter，及び Wassr 用の専用ブラウザであり，RSS リーダ機能も備える統合ブラウザです
  そのため，本アプリを使用するには mixi，Twitter，Wassr のアカウントが必要になります
  # RSS リーダとしてのみ使用する場合はその限りではありません

  mixi を利用するには既存の利用者からの招待が必要です
  Twitter，Wassr は自己登録制ですので，いつでもアカウントを取得することができます
  Twitter はこちらから取得してください
  Wassr はこちらから取得してください

  初回起動時に上記画面が表示されます
  mixi，Twitter 共に使用する場合はそのまま OK を，
  使用しないものがある場合は該当サービスのチェックを外して OK を押下してください

  次にチェックしたサービスのアカウント登録を行います

  mixi の場合はメールアドレスとパスワードを入力してください

  Twitter の場合は ID とパスワードを入力してください
  Wassr の場合も同様に ID とパスワードを入力してください

  次に絵文字ダウンロードの確認画面が表示されます

  「はい」を選んだ場合は以下の画面が表示されます

  連続DL にチェックをしたまま「開始」を押下するとリスト上の全てのファイルをダウンロードします
  チェックを外した場合は選択されている項目のみをダウンロードします

  また，ダウンロード確認画面で「いいえ」を選んだ場合は以下の画面が表示されます

  ここで「はい」を選んだ場合，次回起動時に再びダウンロード確認の画面が表示されます
  「いいえ」を選んだ場合は以後確認画面は表示されなくなります
  ※ ここで「いいえ」選び，絵文字ファイルをダウンロードしない場合，MZ4 上で絵文字の表示ができません
     また，「いいえ」を選んだ後に絵文字ファイルをダウンロードするには ini の編集が必要になります
     編集方法は mz3.ini 編集の【AutoDownloadEmojiImageFiles】を参照してください

  これで初期設定は終了です
  細かい設定についてはこちらをご覧ください
