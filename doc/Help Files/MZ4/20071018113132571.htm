* ヘルプ

　ヘルプやデバッグ情報などを表示します

　●MZ4ヘルプ
　　本ヘルプを表示します
　●改版履歴
　　MZ3.i / MZ4 の改版履歴 ( ChangeLog.txt ) を表示します
　●ソフトウェアの更新を確認
　　新たなバージョンが出ていないか確認します
　●バージョン情報
　　MZ4 のバージョン, ライセンス情報などを確認できます
　　reset ボタンを押下することで今までの総受信バイト数をクリアできます
  ●デバッグ情報
    ・デバッグ情報
      気にしないで下さい
    ・エラーログを開く
      出力されたエラーログ ( mz3log.txt ) を MZ4 上で閲覧します
    ・URLを開く

      指定した URL を選択されたエンコードで表示します
    ・HTMLを開く
      指定した .html ファイルを表示します
      ※ ファイルサイズが大きい場合，表示に時間が掛かります
