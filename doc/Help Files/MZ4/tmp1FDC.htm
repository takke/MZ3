* メインメニューについて

　メインメニューは F1 キー押下で呼び出します

　●画面
　　画面遷移を行った後に操作可能になります
　　【前の画面へ】
　　　画面遷移後に有効になります。遷移前の画面に戻ります
　　【次の画面へ】
　　　遷移先から戻った場合に有効になります。戻る前の画面に進みます

　●設定
　　【一般】
　　　設定画面を開きます。
　　　詳細は「設定画面について」をご覧ください
　　【記事取得】
　　　レポート画面で選択した項目を表示する際に表示する件数を選択します
　　【文字サイズ】
　　　設定画面で設定した文字サイズを選択します
　　　詳細は「設定画面について」の表示タブ, 【文字サイズ】を参照してください
　　【スキン変更】
　　　MZ3.i 用設定項目のため，MZ3 for Windows では使用しません

　●編集
　　読み取り属性の項目 (コメント欄など) ではコピーと全て選択のみ,
　　書き込み可能の項目 (日記投稿画面など) では全てのアイテムを選択できます
　　また, キーボードから Windows PC と同様のショートカットも使用できます
　　※CTRL+C, CTRL+X, CTRL+Z などが使用できます

　●ブックマーク
　　コミュニティトピックスをブックマーク登録します
　　ここで登録されたブックマークは (通常) その他タブにあるブックマーク一覧から遷移できます

　●その他
　　【新着チェック】
　　　新着メッセージ, コメントを取得します
　　【全てを巡回】
　　　巡回予約を行った項目を一括巡回します
　　【未読を巡回】
　　　巡回予約を行った項目のうち, 未読の項目のみ取得します
　　　※画面上で黒表示の項目を読み飛ばします

　●ヘルプ
　　【MZ3 ヘルプ】
　　　MZ3 for Windows 上でヘルプドキュメントを開きます
　　【改版履歴】
　　　MZ3 for Windows 上で ChangeLog.txt を開きます
　　【バージョン情報】
　　　MZ3 for Windows のバージョン, ライセンス情報などを確認できます
　　　reset ボタンを押下することで今までの総受信バイト数をクリアできます
　　【デバッグ情報】
　　　・デバッグ情報
　　　　気にしないで下さい
　　　・エラーログを開く
　　　　出力されたエラーログ ( MZ3\mz3log.txt ) を MZ3 for Windows 上で閲覧します
