　Q.データが表示されない / 表示が更新されない

　A.いくつか可能性が上げられます

　1.処理が実行されていない場合
　　もう一度エンターキーを押下するか，ダブルクリックしてください
　　※上部ペインでは基本的にシングルタップで選択, ダブルタップで実行になります

　2.アクセス禁止
　　データが表示できない症状が出た直前に巡回を行っていた場合，
　　mixi からアクセス制限 ( アカウントの一時停止 ) をされている場合があります
　　MZ3/MZ4 で複数回データ取得を連続して行うとこの制限をされやすくなるので
　　十分に気をつけてください

　3.toppage_group.ini の破損
　　何らかの原因で toppage_group.ini が破損している可能性があります
　　設定 | 一般 | 表示 からタブの初期化を行ってください
