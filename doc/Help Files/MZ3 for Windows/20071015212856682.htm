　A.下記を確認してください
　　●メールアドレス，パスワードは正しく入力されていますか？
　　●MZ4 ではメールアドレス，パスワードを暗号化しているため，
　　　一部文字で暗号化に失敗している場合があります( + 記号など )
　　　このような場合は mixi の登録内容をアルファベットのみのものに変更し,
　　　その変更後のメールアドレス，パスワードで再度試してください
　　●MZ4 のバージョンは最新になっていますか？
　　　mixi が頻繁に仕様変更を行うため，旧バージョンでは正しく動作しない場合があります
　　　Beta 版であっても正常に動作しますので，不具合報告する前に必ず最新のバージョンを試してください

　　また，mixi では短時間に過剰なアクセスを試みたユーザにアクセス禁止を行うことがあります
　　高速ブロードバンド環境で巡回機能などを連続して行うとこの規制が掛かる場合があります
　　十分に注意してご利用ください
