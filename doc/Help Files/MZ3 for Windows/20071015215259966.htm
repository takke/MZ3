* 既知のバグ
  既知のバグ一覧です

 - あしあとが取得できない場合がある（特に起動直後のアクセスの場合に多い）
 - [MZ4] 絵文字の背景が黒になる場合がある
 - 8M以上のファイルをダウンすると落ちる(チャンクの問題があるのでとりあえず保留)
 - ブックマークに登録すると文字が切れたり消えたりする
 - ログを読んでる途中に更新→ログが二重になる→再度更新→落ちる（不具合トピ No.618）
 - スペースキーで半角スペースが入力できない。
 - 半角の＆を含む場合に＆が消える場合がある。
