* Twitter とは？

  Twitter とは，「つぶやく」ことでコミュニケーションをとる一種の SNS サービスです
  "What are you doing?" といった，簡単な文章 ( 140文字まで ) を投稿することで，
  Follow している・されている友人同士で簡単なコミュニケーションを取ることができます

  サービスの詳細は
  http://twitter.com/
  にある，"What?"，"Why?"，"How?" のタブを参照してください
  または Google などで "Twitter" を検索すれば日本語の紹介サイトも多々ありますので，
  そちらを参照してください
