* はじめに

　MZ3 / MZ3.i は Willcom W-ZERO3 シリーズ用の mixi ブラウザです
　この MZ3.i を Windows に改修したものが MZ3 for Windows です

　本ドキュメントに記載のない事柄についての質問は mixi 内コミュニティ

　　MZ3 -Mixi for ZERO3-
　　http://mixi.jp/view_community.pl?id=1198460

　にて質問してください

　リリース履歴に関しては改版履歴（ChangeLog.txt）をご覧下さい


　!! 重要 !!

　●本ドキュメントの筆者と MZ3 / MZ3.i / MZ3 for Windows の作成者は別になります
　　ですので, 本ドキュメントについて作成者に直接問い合わせるのはやめてください
　　必ず上記コミュニティにてお願いします

　●本ドキュメントの複製, 再配布は自由に行っていただいて結構です
　　但し, 変更を加える場合は必ず修正箇所を改定履歴に記述し,
　　その上で上記コミュニティへ必ず報告してください

　●MZ3 / MZ3.i は基本的に WM5 端末での使用を前提に作られています
　　WM6 Professional 端末の実機テストは行っていますが, 完全ではないかも知れません
　　動作する上でクリティカルな症状などがあった場合は
　　コミュニティへの報告をお願いします

　　また，WM6 Standard 端末での動作確認は取れていません
　　これは OS 自体にある程度の制約がかかり，動作が異なるためです
　　SoftBank の X02HT などをお使いの方は注意してください

　●MZ3 for Windows は Windows OS 用アプリです
　　対応 OS は Windows 2000 以降です
　　Windows 9x 系では使用できないと思いますが，頑張ればどうにかなるかもしれません

　●MZ3 / MZ3.i / MZ3 for Windows はフリーソフトウェアです
　　ソフトウェア使用による損害や問題に対して一切の責任も負いかねますので,
　　すべて使用者の責任において使用してください
