　●背景スキンを変更する
　　\MZ3\skin 内に
　　　body.xxx
　　　header.xxx
　　　report.xxx
　　の名称で保存すれば既定以外のものも使用できます
　　\MZ3\skin 以下に適当なディレクトリを作成し, その中に
　　上記名称で保存したファイルを格納することでスキン切り替えの対象となります
　　自作スキンを用意する場合は ini ファイル編集の「skin.ini」を参照してください

　　また, 下記トピックス
　　【公開】こんなスキン作りました！
　　http://mixi.jp/view_bbs.pl?id=22212014&comm_id=1198460
　　にてオリジナルのスキンを募集・公開しています
　　自作スキンを作られた方は是非投稿してください
