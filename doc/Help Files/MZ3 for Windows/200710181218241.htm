* エスケープ / バックスペースキーについて

　エスケープ / バックスペースキーは基本的に同様の動きをします

　●メイン画面
　　上下ペインの行き来をします
　●レポート画面
　　メイン画面に戻ります
　●書き込み画面
　　通常のテキスト入力と同様の動きになります
　●設定画面
　　エスケープキーのみ，画面を閉じます
　　バックスペースキーは使用できません
　●コンテキストメニュー
　　エスケープキーのみ，画面を閉じます
　　バックスペースキーは使用できません

　また，データ通信中にこれらのキーを押下することで通信を中断します
