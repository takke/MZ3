　Q.足あとが10件しか表示されない

　A.仕様です
　　Ver.0.9.0.0 Beta5 より，足あと取得の仕様が変更になり，10件の表示となっています
　　これは，今まで mixi の HTML から取得していたものを mixi の API を使用するようにしたためです
　　今までは独自に HTML を解析していたため，mixi の仕様変更が入る都度に修正が必要でした

　　しかし，API の仕様が変更になることはそうないので，逐次の修正が無用になります
　　ここは保守性とユーザビリティのトレードオフになりますが，
　　「安定したアプリ」を作るために必要な処置ですのでご容赦下さい

　　また，今までのように足あとを20件表示したい場合は Ver.0.8 系をご利用下さい
