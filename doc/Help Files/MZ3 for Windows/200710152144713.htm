* キー操作について

　MZ3 for Windows では上下左右ボタン, エンター，エスケープ，バックスペースを使用します

　●上下ボタン
　　見ている画面にあわせて上下にボタンを操作するとカーソルが移動します
　　(メイン画面のメニューと目次は下端と上端がつながっています)

　●左右ボタン
　　フォーカスのある画面によって以下のように切り替わります

　【メイン画面】
　　・上部ペイン
　　　左：左タブ
　　　右：右タブ
　　・下部ペイン
　　　左：上端・下端
　　　右：名称・時刻の切り替え

　【レポート画面】
　　・上部ペイン
　　　左：上にページスクロール
　　　右：下にページスクロール

　●バックスペース・エスケープ
　　バックスペース・エスケープを押下するとフォーカスのある画面が切り替わります

　●エンターキー
　　カーソル位置のものを選択と決定に使用します
　　選択されていないものを読み込むには2回押下します
　　詳細は「エンターキーについて」を参照してください

　●F1 キー
　　メインメニューを呼び出します
　　詳細は「メインメニューについて」を参照してください

　●F2 キー
　　コンテキストメニューを呼び出します
　　このメニューはフォーカスのある画面によって項目が変化します
　　開かれたメニューは左ボタンで閉じることができます
　　詳細は「F2 キーについて」を参照してください
